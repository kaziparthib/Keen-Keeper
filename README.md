# KeenKeeper 👥

KeenKeeper is a React-based web application designed to help users track, manage, and nurture their most important relationships. Based on the concept of maintaining meaningful social circles, the app provides a clear overview of close friendships, interaction frequencies, detailed timelines, and analytics.

## 🚀 Features & Pages

The application is built around four core views, as designed in Figma:

1. *Dashboard (Friends to keep close in your life)*
   * A comprehensive grid layout displaying your close connections.
   * Quick-glance metrics showing total friends, upcoming catch-ups, and interaction streaks.
   * Status indicators for each friend to keep track of connection health.

2. *Friend Profile View*
   * Detailed personal profile page for individual contacts.
   * Tracks specific milestones, contact frequencies, and customized notes.
   * Social tier and interaction preferences overview.

3. *Timeline & Interaction History*
   * A vertical chronological feed of your past interactions (calls, meetups, messages).
   * Visual logs making it easy to see when you last connected and what was discussed.

4. *Friendship Analytics*
   * Visual data representations (including pie/donut charts) tracking interaction types.
   * Breaks down communication habits (e.g., Call vs. Meetup vs. Text) to help maintain balanced relationships.

---

## 🛠️ Tech Stack

* *Frontend:* React.js
* *Styling:* Tailwind-css
* *Data Visualization:* Chart.js / Recharts (for Friendship Analytics)
* *Icons:* React Icons
